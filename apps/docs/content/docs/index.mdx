A production-ready React component library featuring 40+ UI components, 1500+ icons, multiple built-in themes, and full TypeScript support.

## Features

- **40+ components** — buttons, forms, tables, dialogs, navigation, and more
- **1500+ icons** — comprehensive icon set included
- **Multiple themes** — light, dark, and brand-specific themes out of the box
- **TypeScript-first** — every component is fully typed
- **Accessible** — built on shadcn/ui and Radix UI primitives
- **Zero configuration** — import the styles, use the components

## Quick start

```bash
npm install @acronis-platform/shadcn-uikit tw-animate-css
```

```tsx
import '@acronis-platform/shadcn-uikit/styles';
import { Button } from '@acronis-platform/shadcn-uikit';
```
