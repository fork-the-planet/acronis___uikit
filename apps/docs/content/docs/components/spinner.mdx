## Usage

```tsx
import { Spinner } from '@acronis-platform/ui-react';
```

`Spinner` is a CSS spinning ring with a `role="status"` live region and an
sr-only "Loading…" label. It comes in four `size`s and defaults to the brand
blue; override the color with any `text-*` class. This is a design-pending v1; a
`--ui-spinner-*` tier is pending a Figma pass.
